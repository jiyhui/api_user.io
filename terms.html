서비스 약관

최종 수정일: 2025년 4월 22일

본 서비스 약관("약관")은 사용자가 "APIAuthenticator" 앱(이하 "서비스")을 사용하는 데 적용됩니다.
본 서비스는 개인적인 연구 및 개발 목적으로 Google API에서 데이터를 수집하고 분석하는 도구입니다.

1. 서비스 목적

  • 본 서비스는 Google API를 통해 데이터를 수집하여 개인적으로 활용할 수 있도록 지원합니다.
  • 본 앱은 상업적인 용도가 아닌, 비영리적 개인 프로젝트에 국한됩니다.

2. 사용자 데이터

  • 본 서비스는 사용자로부터 어떠한 개인정보도 직접 수집하지 않습니다.
  • Google OAuth를 통해 인증된 경우, 인증은 Google의 보안 정책을 따릅니다.

3. 제3자 API 사용

  • 본 서비스는 Google API, YouTube Data API 등을 통해 데이터를 수집합니다.
  • 사용자는 Google API의 이용약관 및 정책을 반드시 준수해야 합니다.

4. 책임의 한계

  • 본 서비스는 제공되는 정보의 정확성, 안정성, 지속 가능성에 대해 보장하지 않습니다.
  • 서비스 사용 중 발생하는 문제에 대해 개발자는 책임을 지지 않습니다.

5. 변경 사항

  • 본 약관은 사전 고지 없이 변경될 수 있으며, 변경 시 페이지를 통해 공지됩니다.
